这是一个入门学习安卓开发后的第二个练手项目，目标是实现对接学校教务系统，实时获取课程表（也因为我喜欢用原生程序而不是小程序……）

目前已完成大体结构开发，可以日用，后续更新主要围绕UI美化(Material You)和Bug修复，UI革新后再加新功能
