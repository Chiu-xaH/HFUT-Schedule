【肥工教务通】

这是一个入门学习安卓开发后的第二个练手项目，目标是实现对接合肥工业大学学生教务系统，实时获取课程表（也因为我喜欢用原生程序而不是小程序……）(现在对接了信息门户,指间工大等等,能查的东西越来越多,越来越实用)
